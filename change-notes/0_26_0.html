0.26.0 (09/12/2025)

This release contains the following fixes:

  • Editor completion does not work in 2025.3 EAP6 (#158, #159)

Note that starting from 0.26.0, the minimum supported IDEA version is 2025.3 (253.17525).
The goal has always been to maintain compatibility with as many versions as possible (down to 2023.3 for 0.25.0). However, the amount of manual testing and hacks required to do so does not make sense anymore, considering all the breaking changes shipped between 2024.x and 2025.x.
