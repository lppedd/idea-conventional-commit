0.10.0 (06/04/2020)

Fifteenth alpha release.
New features

  • Provided a new inspection to highlight where the commit message doesn't follow the Conventional Commit standard.
    The inspection is enabled by default so you can try it out, but you may disable it via Version Control > Commit > Commit message inspections
  • Provided a new experimental Extension Point to contribute with inspections: commitInspectionProvider
